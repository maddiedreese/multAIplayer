# CLI-160: Signed CLI packaging and release isolation

State: waiting_for_orchestrator_approval

## Outcome

Produce independently versioned, signed Apple-silicon macOS CLI archives and
checksums without entering or changing the desktop updater/artifact contract.

## Allowed scope

- CLI packaging, signing, checksums, notices, release manifest, and documentation.
- Shared release helpers only through an explicitly reviewed additive interface.

## Acceptance criteria

- [ ] Installed executable is `multAIplayer`.
- [ ] CLI version/artifacts are distinct from desktop versions/artifacts.
- [ ] Desktop updater, notarization, assets, and synchronized versions are unchanged.
- [ ] Reproducible source revision, checksum, notices, and installation steps exist.
- [ ] CLI publication remains manual until owner approval.

## Verification

- Clean-machine package smoke test, signature verification, checksum verification,
  notices/license checks, and desktop release-contract verification.
