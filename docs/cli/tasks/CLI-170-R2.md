# CLI-170-R2: Join prompt and root install discoverability

State: owner_approved
Plan version: 1.1

## Outcome

Make the public install command directly discoverable in the root README and
make stdin-only invitation entry complete with a single pasted line and Return,
without requiring an undocumented EOF keystroke.

## Dependencies

- CLI-170-R1 accepted installer and release documentation.
- Owner approval on 2026-07-19.

## Allowed scope

- Root and CLI quickstart documentation.
- The CLI joiner's fixed local prompt and bounded one-line stdin reader.
- Focused CLI tests and semantic documentation checks.
- CLI status evidence.

## Protected scope

- Invitation encoding, entropy, expiry, admission authority, relay APIs, wire
  records, and MLS policy.
- Desktop behavior, release metadata/workflows, Cargo.lock, updater, signing,
  notarization, versions, and asset manifests.
- Dependencies, publication, tags, release assets, and signing identities.

## Acceptance criteria

- [ ] The exact public install command is visible in the root README.
- [ ] `multAIplayer room join` prints a fixed local prompt, reads exactly one
      bounded line, and proceeds after Return without waiting for EOF.
- [ ] The invitation remains absent from shell arguments, logs, diagnostics,
      room events, and fixed prompt output.
- [ ] Empty, whitespace-bearing, malformed, and oversized input still fails
      closed through the existing admission parser.
- [ ] Desktop release and protocol behavior remain unchanged.

## Required verification

- CLI formatting and warnings-denied Clippy.
- Focused join prompt/input tests and the production admission journey.
- Locked CLI/protocol/mixed-client suite.
- Installer policy and documentation checks.
- Changed-path classification/release-isolation tests and exact protected-path
  review.

## Stop conditions

Stop if the correction requires a dependency, hidden terminal-input library,
invitation format or authority change, relay/MLS change, desktop change,
protected release change, signing credential, or publication.
