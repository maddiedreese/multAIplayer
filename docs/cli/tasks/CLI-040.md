# CLI-040: GitHub authentication and secure device identity

State: active

## Outcome

Authenticate through the existing GitHub device flow, restore origin-bound relay
sessions, and initialize persistent signing/HPKE identity in macOS Keychain.

## Allowed scope

- CLI auth/device modules and platform adapters.
- Additive extraction of proven native identity behavior after focused review.

## Acceptance criteria

- [ ] Login works with printed URL/code and optional trusted URL opening.
- [ ] Session restoration is origin bound.
- [ ] Tokens and private keys never appear in args, logs, or diagnostics.
- [ ] Logout does not silently delete room keys.
- [ ] Device registration is compatible with desktop devices.

## Verification

- Mock device-flow and restoration integration tests.
- Keychain error and secret-redaction tests.
